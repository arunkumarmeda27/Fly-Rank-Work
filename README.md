# Fly Rank Capstone Project

This repository serves as the capstone project for the Fly Rank track, focusing on AI-assisted development, high-fidelity UI design, and modern web application development best practices.

## 🚀 Technology Stack
- **Core Structure**: HTML5 (Semantic elements)
- **Frontend Logic**: Vanilla JavaScript (ES6+ standard)
- **Styling**: Vanilla CSS (Modern CSS variables, Flexbox/Grid, Responsive layouts)
- **Runtime & Toolchain**: Node.js (LTS)

## 🛠️ Getting Started

### Prerequisites
Make sure you have Node.js (LTS version) and Git installed on your system.

### Setup
Clone this repository and install any dependencies:
```bash
git clone https://github.com/arunkumarmeda27/Fly-Rank-Work.git
cd Fly-Rank-Work
npm install
```

### Development
To run the local development server:
```bash
npm run dev
```

### Production Build
To compile and optimize the project for production:
```bash
npm run build
```
